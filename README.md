## Features
- User Management
- Site Management
- Charities Management
- Logo Set Management
- Banner Management
- Custom Form Designer
- Submitted forms management with CSV export
- - Soft delete when exported
- - Cleanup after 7 days
- - Encrypted forms
- Self Signed certificate authentication
